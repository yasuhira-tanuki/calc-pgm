# calc-pgm

プログラミング用計算機。整数・浮動小数点の算術演算とビット演算をサポートし、結果を DEC/HEX/OCT/BIN の各形式で表示します。単位変換関数・`conv()` による単位間変換も利用できます。

## ビルドと実行

```bash
make                        # ビルド
./calc-pgm                  # 対話モードで起動
./calc-pgm -e "1 + 2"      # 引数モードで式を評価して終了
./calc-pgm -b "0xFF & 0x0F" # ビット演算を評価して終了
./calc-pgm -l 1024          # ln / log2 / log10 を一括表示
./calc-pgm -f hex 255       # 指定フォーマットで表示
./calc-pgm -u 2 gib mb      # 単位変換
./calc-pgm -t               # 整数型・浮動小数点型の一覧を表示
./calc-pgm -c               # 文字コードの一覧を表示
make clean                  # 成果物を削除
```

## 使い方

起動するとプロンプト `>` が表示され、式を入力して Enter で評価します。

```
> 0xFF & 0x0F
  DEC: 15
  HEX: 0xF
  OCT: 0o17
  BIN: 0b1111

> 3.14 * 2
  = 6.28

> gib(2)
  = 2147483648

> conv(2, gib, mb)
  = 2147.483648
```

## 演算子

| 種別 | 演算子 | 備考 |
|------|--------|------|
| 算術 | `+ - * / %` | `%` は整数のみ |
| べき乗 | `**` | 右結合。整数はオーバーフロー検出あり |
| ビット | `& \| ^ ~ << >>` | 整数のみ、シフトは 0〜63 |
| 単項 | `- + ~` | `~` は整数のみ |
| 括弧 | `( )` | |

演算子優先順位 (低→高): `|` → `^` → `&` → `<<,>>` → `+,-` → `*,/,%` → 単項 → `**` → 一次式

## 数値リテラル

| 形式 | 例 |
|------|----|
| 10進整数 | `42`, `-42` |
| 16進 | `0xFF`, `0x2A` |
| 2進 | `0b1010` |
| 8進 | `0o52` |
| 浮動小数点 | `3.14`, `1e10` |
| 前回の結果 | `_` |

浮動小数点の表示は小数点表記のみ (科学的記数法なし)。例: `0.000001`

## 対数関数

引数は正の数のみ。結果は常に浮動小数点で返ります。

| 関数 | 意味 |
|------|------|
| `log(x)` | 自然対数 (底 e) |
| `log2(x)` | 2を底とする対数 |
| `log10(x)` | 10を底とする対数 |

```
> log2(1024)
  = 10

> log(1)
  = 0

> log10(1000)
  = 3
```

## 表示変換関数

特定の形式のみ表示したい場合に使います。引数には任意の式を指定できます。整数のみ対応。

| 関数 | 表示 |
|------|------|
| `dec(式)` | 10進のみ (`= N` 形式) |
| `hex(式)` | 16進のみ |
| `oct(式)` | 8進のみ |
| `bin(式)` | 2進のみ |

```
> hex(255)
  HEX: 0xFF

> bin(1 << 4)
  BIN: 0b1_0000

> dec(0b1100 | 0b0011)
  = 15
```

## 単位変換関数

引数に任意の式を指定できます。計算結果は10進のみ表示 (`= N` 形式)。

### データサイズ (戻り値: バイト)

| 関数 | 換算 |
|------|------|
| `kib(n)` | n × 1,024 |
| `mib(n)` | n × 1,024² |
| `gib(n)` | n × 1,024³ |
| `tib(n)` | n × 1,024⁴ |
| `kb(n)` | n × 1,000 |
| `mb(n)` | n × 1,000² |
| `gb(n)` | n × 1,000³ |
| `tb(n)` | n × 1,000⁴ |

### 周波数 (戻り値: Hz)

| 関数 | 換算 |
|------|------|
| `hz(n)` | n × 1 |
| `khz(n)` | n × 1,000 |
| `mhz(n)` | n × 1,000,000 |
| `ghz(n)` | n × 1,000,000,000 |

### 時間 (戻り値: 秒)

| 関数 | 換算 |
|------|------|
| `ns(n)` | n ナノ秒 (n × 1e-9 秒) |
| `us(n)` | n マイクロ秒 (n × 1e-6 秒) |
| `ms(n)` | n ミリ秒 (n × 1e-3 秒) |
| `sec(n)` | n 秒 |

### 単位間変換: conv()

```
conv(値, 変換前単位, 変換後単位)
```

同カテゴリ内の単位間で変換します。異カテゴリはエラー。

| カテゴリ | 基準単位 | 使用できる単位 |
|----------|----------|----------------|
| データサイズ | `b` | `b` `kib` `mib` `gib` `tib` `kb` `mb` `gb` `tb` |
| 周波数 | `hz` | `hz` `khz` `mhz` `ghz` |
| 時間 | `sec` | `ns` `us` `ms` `sec` |

```
> conv(2, gib, mb)
  = 2147.483648

> conv(500, ms, us)
  = 500000

> conv(100, mhz, ghz)
  = 0.1

> conv(2, gib, b)
  = 2147483648

> conv(2, gib, mhz)
  エラー: 構文エラー: 単位のカテゴリが異なる
```

### 使用例

```
> gib(2)
  = 2147483648

> gib(2) * 512
  DEC: 1099511627776
  HEX: 0x10000000000
  OCT: 0o20000000000000
  BIN: 0b1_0000_0000_0000_0000_0000_0000_0000_0000_0000_0000

> ms(500)
  = 0.5

> sec(1) / ms(1)
  = 1000
```

## コマンドライン引数

| オプション | 動作 |
|------------|------|
| `-e <式>` | 式を演算して計算結果を表示し終了 |
| `-b <式>` | ビット演算式を評価して結果を表示し終了 |
| `-l <式>` | 式を評価し ln / log2 / log10 を一括表示して終了 |
| `-f <形式> <式>` | 指定フォーマットで結果を表示し終了 |
| `-u <値> <変換前> <変換後>` | 単位変換して結果を表示し終了 |
| `-t` | 整数型・浮動小数点型の一覧を表示し終了 |
| `-c` | 文字コードの一覧を表示し終了 |

引数に含まれるスペースはクォートで囲むか、そのまま並べて渡すことができます。

```bash
$ ./calc-pgm -e "0xFF & 0x0F"
  DEC: 15
  HEX: 0xF
  OCT: 0o17
  BIN: 0b1111

$ ./calc-pgm -e "gib(2)"
  = 2147483648

$ ./calc-pgm -e 1 + 2
  DEC: 3
  HEX: 0x3
  OCT: 0o3
  BIN: 0b11
```

### -l オプション: 対数を一括表示

式を評価した値に対して ln (自然対数)・log2・log10 を一括表示します。引数には任意の式を指定できます。引数が正の数でない場合はエラーになります。

```bash
$ ./calc-pgm -l 1024
  ln   : 6.9314718056
  log2 : 10
  log10: 3.0102999566

$ ./calc-pgm -l 2.718281828
  ln   : 0.9999999998
  log2 : 1.4426950406
  log10: 0.4342944818

$ ./calc-pgm -l "2 ** 10"
  ln   : 6.9314718056
  log2 : 10
  log10: 3.0102999566
```

### -f オプション: フォーマット指定表示

式を評価し、指定したフォーマットで結果を表示します。

| 形式 | 表示 |
|------|------|
| `dec` | 10進のみ |
| `hex` | 16進のみ |
| `oct` | 8進のみ |
| `bin` | 2進のみ |
| `all` | DEC/HEX/OCT/BIN 全表示 |

`hex` / `oct` / `bin` は整数のみ対応。浮動小数点に指定するとエラーになります。

```bash
$ ./calc-pgm -f hex 255
  HEX: 0xFF

$ ./calc-pgm -f bin 255
  BIN: 0b1111_1111

$ ./calc-pgm -f oct 255
  OCT: 0o377

$ ./calc-pgm -f dec 0xFF
  = 255

$ ./calc-pgm -f all 255
  DEC: 255
  HEX: 0xFF
  OCT: 0o377
  BIN: 0b1111_1111

$ ./calc-pgm -f hex "0xFF & 0x0F"
  HEX: 0xF

$ ./calc-pgm -f bin "1 << 4"
  BIN: 0b1_0000
```

### -b オプション: ビット演算

`-b` はビット演算専用のオプションです。出力形式は対話モードと同等です。

```bash
$ ./calc-pgm -b "0xFF & 0x0F"
  DEC: 15
  HEX: 0xF
  OCT: 0o17
  BIN: 0b1111

$ ./calc-pgm -b "1 << 8"
  DEC: 256
  HEX: 0x100
  OCT: 0o400
  BIN: 0b1_0000_0000

$ ./calc-pgm -b "0b1100 ^ 0b1010"
  DEC: 6
  HEX: 0x6
  OCT: 0o6
  BIN: 0b110

$ ./calc-pgm -b "~0"
  DEC: -1
  HEX: 0xFFFFFFFFFFFFFFFF
  OCT: 0o1777777777777777777777
  BIN: 0b1111_1111_1111_1111_1111_1111_1111_1111_1111_1111_1111_1111_1111_1111_1111_1111
```

### -u オプション: 単位変換

`conv()` 関数と同等の単位変換をコマンドラインから直接実行します。

```
calc-pgm -u <値> <変換前単位> <変換後単位>
```

同カテゴリ内の単位間でのみ変換できます。異カテゴリ間はエラーになります。

```bash
$ ./calc-pgm -u 2 gib mb
  = 2147.483648

$ ./calc-pgm -u 500 ms us
  = 500000

$ ./calc-pgm -u 100 mhz ghz
  = 0.1

$ ./calc-pgm -u 2 gib b
  = 2147483648
```

値の部分には式も指定できます。

```bash
$ ./calc-pgm -u 1+1 gib mb
  = 2147.483648
```

## コマンド

| コマンド | 動作 |
|----------|------|
| `help` | ヘルプを表示 |
| `types` | 整数型・浮動小数点型の一覧 (サイズ・値の範囲・有効桁数) を表示 |
| `enc` | 文字コード一覧 (ASCII, UTF-8/16/32, Shift-JIS, EUC-JP) を表示 |
| `size <文字列>` | 文字数と各エンコードのバイト数を表示 |
| `quit` / `exit` | 終了 |

## アーキテクチャ

```
src/main.c     REPLループ
src/lexer.c    文字列 → トークン列
src/parser.c   再帰下降構文解析、calc_eval が公開 API
src/display.c  Value を各形式で出力
include/       各モジュールのヘッダ
build/         コンパイル成果物 (自動生成)
```

- `Value` 型は `is_float` フラグで整数 (`int64_t`) と浮動小数点 (`double`) を区別します
- ビット演算・シフト・剰余は整数のみ許可し、float に適用するとエラーになります
- 単位変換関数の結果は常に10進表示 (`= N`)。他の演算と組み合わせると FMT_ALL に戻ります
